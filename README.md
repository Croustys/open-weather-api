# Open Weather API UI practice
Built using Snowpack, React, Typescript  
also learning react-bootstrap

## Installation
Clone this repo
```bash
 git clone https://github.com/Croustys/2021_kadi_frontend.git
```

Install dependencies
```bash
 npm install
```

Start up dev server
```bash
 npm start
```
